# count

The `count` function takes an object `opts` as an argument and returns the number documents in the database. The `opts` object can have a property `listQueryFn`, which is a function that can be used to customize the query. If `listQueryFn` is not present, the function will use a default query.

## Usage

```js
import countCustomers from "path/to/data/customer/count";

const count = await countCustomers();
```

## Parameters

- `opts` (object, optional): An options object for configuring the behavior of the function.
    - `listQueryFn` (function, optional): A function that can be used to customize the query.

## Returns

- `count` (number): The number of documents in the database.
